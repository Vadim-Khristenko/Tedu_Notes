**–ß—Ç–æ —ç—Ç–æ:** —ç—Ç–∞–ø, –Ω–∞ –∫–æ—Ç–æ—Ä–æ–º –∏–Ω—Å—Ç—Ä—É–º–µ–Ω—Ç –ø—ã—Ç–∞–µ—Ç—Å—è —Ä–∞–∑—Ä–µ—à–∏—Ç—å –∏–º–µ–Ω–∞ –∏ —Ç–∏–ø—ã ‚Äî –ø–æ–Ω—è—Ç—å, –∫–∞–∫–∞—è –∏–º–µ–Ω–Ω–æ —Ñ—É–Ω–∫—Ü–∏—è –∏–ª–∏ –ø–µ—Ä–µ–º–µ–Ω–Ω–∞—è –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è.

**–ß—Ç–æ —ç—Ç–æ:** –ø—Ä–µ–¥—Å—Ç–∞–≤–ª–µ–Ω–∏–µ –≤–æ–∑–º–æ–∂–Ω—ã—Ö –ø—É—Ç–µ–π –≤—ã–ø–æ–ª–Ω–µ–Ω–∏—è –ø—Ä–æ–≥—Ä–∞–º–º—ã –≤ –≤–∏–¥–µ –≥—Ä–∞—Ñ–∞ ‚Äî —É–∑–ª—ã —ç—Ç–æ –±–∞–∑–æ–≤—ã–µ –±–ª–æ–∫–∏ –∫–æ–º–∞–Ω–¥, —Ä—ë–±—Ä–∞ ‚Äî –ø–µ—Ä–µ—Ö–æ–¥—ã.

**–ü—Ä–∏–º–µ—Ä:** –ø—Ä–∞–≤–∏–ª–æ "–≤–æ–∑–º–æ–∂–Ω—ã–π null-dereference (–Ω—É–ª–µ–≤–æ–π —É–∫–∞–∑–∞—Ç–µ–ª—å)" –±—É–¥–µ—Ç —Å–º–æ—Ç—Ä–µ—Ç—å –Ω–∞ –≤—Å–µ –ø—É—Ç–∏, –≤–µ–¥—É—â–∏–µ –∫ —Ç–æ—á–∫–µ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏—è —É–∫–∞–∑–∞—Ç–µ–ª—è.

**–ü—Ä–∏–º–µ—Ä CFG –¥–ª—è —É—Å–ª–æ–≤–Ω–æ–≥–æ –æ–ø–µ—Ä–∞—Ç–æ—Ä–∞:**

**–ß—Ç–æ —ç—Ç–æ:** –∞–Ω–∞–ª–∏–∑ —Ç–æ–≥–æ, –∫–∞–∫ –¥–∞–Ω–Ω—ã–µ (–∑–Ω–∞—á–µ–Ω–∏—è –ø–µ—Ä–µ–º–µ–Ω–Ω—ã—Ö) —Ç–µ–∫—É—Ç –ø–æ –ø—Ä–æ–≥—Ä–∞–º–º–µ; **SSA** (Static Single Assignment) ‚Äî –ø—Ä–µ–¥—Å—Ç–∞–≤–ª–µ–Ω–∏–µ, –≥–¥–µ –∫–∞–∂–¥–∞—è –ø–µ—Ä–µ–º–µ–Ω–Ω–∞—è –ø—Ä–∏—Å–≤–∞–∏–≤–∞–µ—Ç—Å—è —Ä–æ–≤–Ω–æ –æ–¥–∏–Ω —Ä–∞–∑, —á—Ç–æ —É–ø—Ä–æ—â–∞–µ—Ç –∞–Ω–∞–ª–∏–∑.

**–ö–∞–∫ —Ä–µ–∞–ª–∏–∑—É–µ—Ç—Å—è:** —Å—Ç—Ä–æ—è—Ç—Å—è —Ç–∞–±–ª–∏—Ü—ã –æ–ø—Ä–µ–¥–µ–ª–µ–Ω–∏—è/–∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏—è (def-use), –≤—ã—á–∏—Å–ª—è—é—Ç—Å—è –∂–∏–≤–æ—Å—Ç–∏ –ø–µ—Ä–µ–º–µ–Ω–Ω—ã—Ö, —Å—Ç—Ä–æ–∏—Ç—Å—è SSA –ø—Ä–∏ –Ω–µ–æ–±—Ö–æ–¥–∏–º–æ—Å—Ç–∏.

**–û–≥—Ä–∞–Ω–∏—á–µ–Ω–∏—è:** –¥–æ—Ä–æ–≥–æ –ø–æ —Ä–µ—Å—É—Ä—Å–∞–º; –∏–Ω–æ–≥–¥–∞ –ø—Ä–∏–±–µ–≥–∞—é—Ç –∫ —ç–≤—Ä–∏—Å—Ç–∏–∫–∞–º, —á—Ç–æ–±—ã —É—Å–∫–æ—Ä–∏—Ç—å –∞–Ω–∞–ª–∏–∑ –Ω–∞ –±–æ–ª—å—à–∏—Ö –∫–æ–¥–æ–≤—ã—Ö –±–∞–∑–∞—Ö.

class MyPrintfCheck : public ClangTidyCheck {
public:
    MyPrintfCheck(StringRef Name, ClangTidyContext *Context)
        : ClangTidyCheck(Name, Context) {}
    
    void registerMatchers(MatchFinder *Finder) override {
        Finder->addMatcher(
            callExpr(callee(functionDecl(hasName("printf"))))
                .bind("printfCall"),
            this
        );

        const auto *Call = Result.Nodes.getNodeAs<CallExpr>("printfCall");

            diag(Call->getBeginLoc(), 
                "Use fmt::print instead of printf");
        }
    }
};
